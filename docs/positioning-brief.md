## FAQs I'll get

**"Why n8n over Zapier / Make?"**
Zapier gets expensive (per-task billing). Make hits ceilings. n8n is self-hostable, 1,200+ integrations, custom code support, complex workflows without breaking. For anything beyond simple triggers, n8n is cheaper and more flexible long-term.

**"What does it cost to run after you're done?"**
Self-hosted n8n: ~$6/month for a VPS. LLM costs depend on volume — $0 if we go local Ollama, $20–200/month for cloud APIs on SMB workloads. All costs logged, you'll have a dashboard.

**"Can we keep data off OpenAI?"**
Yes. I can deploy using Ollama on your own infrastructure — LLM runs locally, no data ever leaves your network. Useful for healthcare, legal, financial, or IP-sensitive workflows.

**"What if something breaks after delivery?"**
Every package includes post-launch support. Monitoring baked in — Telegram/Slack alerts on errors, not silent failures. Monthly retainers available for ongoing maintenance.

**"How do I know you won't disappear?"**
Everything is in your GitHub, your infrastructure, your accounts. No lock-in to me. If I vanish, any competent n8n developer can pick it up because documentation is client-grade.
